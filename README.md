# Ember & Bean Coffee Co.

A modern custom coffee shop website built with Next.js, React, TypeScript, and Tailwind CSS.

## Project Status

The site now includes a complete static marketing experience for Ember & Bean Coffee Co. with:

- A premium homepage with hero, featured drinks, brand philosophy, and footer
- A full menu page with Philippine peso pricing
- An about page with brand story, values, and daily rhythm sections
- A contact page with local Philippine location details, call/email actions, and order-ahead messaging
- Responsive navigation with a mobile menu

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Lucide React
- Framer Motion
- GSAP

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```
